## How to Use Templates

### 1. Browse Templates
Visit `/templates` to see all available templates with previews.

### 2. Select a Template
Click "Use Template" on any template to start creating your page.

### 3. Customize Content
In the editor:
- Click any section to edit
- Change text in the right sidebar
- Switch between Arabic and English
- Customize colors and fonts

### 4. Publish
Click "Publish" when ready to make your page live.

```typescript
    language: 'en' | 'ar'
  }
}
```

## Design Principles

All templates follow modern landing page best practices:

4. **Mobile-First** - Responsive design for all devices
5. **Conversion Focused** - Strategic CTA placement
6. **Professional Aesthetics** - Modern, clean, trustworthy designs

## Customization Tips

### Colors
- Primary color: Main brand color for buttons and accents
- Secondary color: Supporting color for variety
- Background: Section backgrounds
- Text: Content text color
- Accent: Highlights and special elements

**Arabic Fonts Available:**
