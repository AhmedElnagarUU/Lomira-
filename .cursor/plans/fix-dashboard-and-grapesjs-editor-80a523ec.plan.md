# Redesign Custom Builder Templates

## Overview

Completely redesign all 4 templates in the custom builder with modern, professional, and conversion-focused designs. Each template should have polished UI, better typography, proper spacing, engaging visuals, and effective layouts.

## Design Principles

### 1. Typography

- Use proper font size hierarchy (h1: 48-64px, h2: 36-48px, body: 16-18px)
- Better line heights (1.2-1.6)
- Proper font weights (bold for headings, regular for body)
- Improved readability with proper contrast

### 2. Spacing & Layout

- Consistent padding (60-120px for sections)
- Proper margins between elements
- Better use of whitespace
- Responsive grid systems
- Max-width containers (1200px)

### 3. Visual Design

- High-quality Unsplash images
- Better image aspect ratios
- Subtle shadows and borders
- Smooth transitions and hover effects
- Modern color schemes (teal, blue, purple gradients)
- Professional button styles

### 4. Components

- Better form designs with labels
- Improved card layouts
- Professional pricing tables
- Engaging CTA sections
- Better feature showcases

## Design Specifications

### Color Palette

- Primary: #06b6d4 (Cyan)
- Secondary: #3b82f6 (Blue)
- Accent: #8b5cf6 (Purple)
- Dark: #1e293b (Slate)
- Light: #f8fafc (Slate light)
- Text: #1e293b (Dark), #64748b (Gray)

### Typography Scale

- Hero H1: 56-64px, bold
- Section H2: 42-48px, bold
- Card H3: 24-28px, semibold
- Body: 16-18px, regular
- Small: 14px, regular

### Spacing Scale

- Section padding: 80-120px vertical, 20px horizontal
- Card padding: 40px
- Element gaps: 20-40px
- Container max-width: 1200px
